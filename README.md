# [Core]PlayerSettingsNext
ClientCookies analogue for CS2. API is compatible with the basic version: [PlayerSettingsCS2](https://github.com/NickFox007/PlayerSettingsCS2/)

## Required packages:
1. [CounterStrikeSharp](https://github.com/roflmuffin/CounterStrikeSharp/)
2. [CS2-AnyBaseLibNext](https://github.com/darkerz7/CS2-AnyBaseLibNext/)

## Installation:
1. Compile or copy PlayerSettings to `counterstrikesharp/plugins/PlayerSettings` folger
2. Compile or copy PlayerSettingsApi to `counterstrikesharp/shared/PlayerSettingsApi` folger
3. Restart server
